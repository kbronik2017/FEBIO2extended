void FETiedBiphasicSurface::UpdateNodeNormals ( )

calculate the nodal normals

This function calculates the node normal. Due to the piecewise continuity of the surface elements this normal is not uniquely defined so in order to obtain a unique normal the normal is averaged for each node over all the element normals at the node
