Detailed Description

This class implements the idea of a level structure.

A level structures groups the nodes per level. The basic notion is that (1) each node in level 1 is adjacent to a node in level 1 and/or level 2. (2) each node in level k is adjacent to a node in level k and/or level k-1 (3) each node in level 1<i<k is adjacent to a node in level i-1, i, or i+1 The width of the level structure is the largest width of any level.

The level structure is used by the node - reorder algorithm implemented in the FENodeReorder class.
