Detailed Description

Uncoupled elastic mixtures.

This class describes a mixture of uncoupled elastic solids. The user must declare uncoupled elastic solids that can be combined within this class. The stress and tangent tensors evaluated in this class represent the sum of the respective tensors of all the solids forming the mixture.
