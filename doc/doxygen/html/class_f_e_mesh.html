int FEMesh::FindInvertedElements ( )
protected

Look for any inverted elements.

Find all elements that inverted. That is, the elements whose jacobian with respect to the reference configuration is negative. Negative initial jacobians may indicate a problem with the mesh (e.g. incorrect node numbering).
