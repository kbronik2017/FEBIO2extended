vec3d &  q,

Project a node onto a surface.

This is slightly modified version of the ClosestPointProjection where the node to be projected is part of this surface. This is used in some contact algorithms that support self-contact. The algorithm is modified in two ways. First, a search_radius limits the max distance for consideration of closest point. Second, the search point cannot be part of the net of the closest point.
