Detailed Description

This class implements a nonlinear explicit solver for solid mechanics problems.

Member Function Documentation

bool FEExplicitSolidSolver::InitEquations ( )
virtual

assemble the element residual into the global residual

Determine the number of linear equations and assign equation numbersThis function initializes the equation system. It is assumed that all free dofs up until now have been given an ID >= 0 and the fixed or rigid dofs an ID < 0. After this operation the nodal ID array will contain the equation number assigned to the corresponding degree of freedom. To distinguish between free or unconstrained dofs and constrained ones the following rules apply to the ID array: 

      /
     |  >=  0 --> dof j of node i is a free dof

ID[i][j] < == -1 –> dof j of node i is a fixed (no equation assigned too) | < -1 –> dof j of node i is constrained and has equation nr = -ID[i][j]-2 \

Implements FESolver.
