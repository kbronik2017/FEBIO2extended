Detailed Description

This class implements the concept of a loadcurve.

A loadcurve is basically a discretized function of time versus load, where load can be interpreted differently in different contexts. The loadcurve stores the (time,load)-pairs for fixed points, which are input from the input file. In between timesteps, the loadcurve class interpolates the (time,load) data pairs according to the interpolation function.
